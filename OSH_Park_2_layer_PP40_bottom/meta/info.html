PP-40 project box Bottom PCB

This is a template with board outline for the PP-40 project box bottom PCB.

The PP-40 project box can contain two boards.

  1. The bottom PCB with 2 smaller holes on top for strength and fixture and one mounting screw hole. It contains two 1.5mm header connectors to insert turned pins for a more compact mounting method.

  2. The top PCB has 2 larger holes on top for the 2 mounting rods. It contains two 1.0mm holed 12 pin headers for turned pins.

Based on the OSH park 2 layer template.
