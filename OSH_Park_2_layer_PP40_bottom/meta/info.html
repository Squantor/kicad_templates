PB1 project box Bottom PCB

This is a template with board outline for the PB1 project box bottom PCB.

The PB1 project box can contain two boards. The bottom PCB with 2 smaller holes on top for strength and fixture and one mounting screw hole. The top PCB has 2 larger holes on top for the 2 mounting rods.

Based on the OSH park 2 layer template.
